# Epub Library
## Getting Started
```
git clone https://github.com/RogZr0/epub_library.git
cd epub_library
python -m venv venv
```

## Activate Virtual Environment
### Windows
```
venv\Scripts\activate
```
### Linux
```
source venv/bin/activate
```

## Dependencies
```
pip install Flask Pillow ebooklib
```

## Run

```
python app.py
```
